# 배민

React 기반 대회 관리 웹 페이지입니다. 대회 생성, 참가 신청, 대진표 확인, 로그인, 회원가입, 마이페이지 흐름을 한 프로젝트 안에서 확인할 수 있습니다.

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000`으로 접속하면 개발 화면을 볼 수 있습니다.

## 주요 기능

- 로그인 및 회원가입
- 마이페이지와 대회 현황 확인
- 대회 생성 및 목록 확인
- 리그전/토너먼트 대진표 확인
- 로컬스토리지 기반 데이터 저장

## 빌드

```bash
npm run build
```
